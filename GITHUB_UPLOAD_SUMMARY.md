## 🚀 **Next Steps for Deployment**

### **1. Clone Repository**
```bash
git clone https://github.com/lilnande7/pengaduan-api.git
cd pengaduan-api
```

### **2. Install Dependencies**
```bash
composer install
npm install
```

### **3. Environment Setup**
```bash
cp .env.example .env
php artisan key:generate
```

### **4. Configure WhatsApp**
```env
FONNTE_TOKEN=your_fonnte_token_here
ADMIN_PHONE=628xxxxxxxxxx
```

### **5. Database Setup**
```bash
touch pengaduan_db
php artisan migrate
php artisan db:seed
```

### **6. Start Server**
```bash
php artisan serve
```
